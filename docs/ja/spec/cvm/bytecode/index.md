## ヘッダ部分

ヘッダ長は 128 バイトである。

|開始位置|サイズ ( バイト )|内容|その他|
|:-:|:-:|:-:|:-:|
|0|8|Ches マジックナンバー||
|8|4|Ches バージョン情報||
|12|4|最終変更日時|形式は Unix タイムスタンプ|
|16|32|作成者||
|||||
