## ホームディレクトリ

以下はOSごとのデフォルトパスである。

### Windows

```
C:\Program Files\Ches\
```

### macOS

```
/Users/Shared/Library/Ches/
```

### Linux

```
/lib/Ches/
```

## ディレクトリ構造

```
root/
  ├ bin/
  ├ config/
  ├ docs/
    ├ vers/
```
