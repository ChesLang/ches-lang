## ノード一覧

|名称|内容|
|:-:|:-:|
|Unknown|不明|
|Root|構文木のルート|
|Args|引数|
|DefFunc|関数定義|
|CallFunc|関数呼び出し|
|DefVar|変数定義|
|InitVar|変数初期化|
|Var|変数|
|If|条件分岐|
|Else|条件分岐 (else)|
|ElseIf|条件分岐 (else if)|
|Loop|繰り返し|
|Express|数式 / 論理式|
|Literal|リテラル|

### Root

#### 子ノード

|:-:|:-:|:-:|

### Args

#### 子ノード

##### 関数定義

|番号|種類|内容|
|:-:|:-:|:-:|

|番号|種類|内容|
|:-:|:-:|:-:|
|- \*1|[Value] \*3|引数|

### Express

#### トークン

|番号|内容|
|:-:|:-:|
|1|演算子|

#### 子ノード

|番号|種類|内容|
|:-:|:-:|:-:|
|1|[Value] \*3|左項 / 左辺|
|2|[Value] \*3|右項 / 右辺|

### Literal

#### トークン

|番号|内容|
|:-:|:-:|
|1|型名|
|2|値|

\*1 ノード数や順は問わない
<br>
\*2 全ノード種のいずれか
<br>
\*3 Var / Express / Literal のいずれか
