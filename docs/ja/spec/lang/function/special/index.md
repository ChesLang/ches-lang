# 特別関数

[ドキュメント](../../../../../index.md) > [日本語](../../../../index.md) > [仕様書](../../../index.md) > [言語仕様書](../../index.md) > [関数](../index.md) > [特別関数](./index.md)

## 概要

特殊な用途に用いる関数で、関数名が `@` から始まる。

## 一覧

|関数名|説明|
|:-:|:-:|
|`@const`|コンストラクタ|
|`@dest`|デストラクタ|
|`@entry`|エントリポイント|
