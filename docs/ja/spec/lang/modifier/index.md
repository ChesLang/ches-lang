# 修飾子

[ドキュメント](../../../../index.md) > [日本語](../../../index.md) > [仕様書](../../index.md) > [言語仕様書](../index.md) > [修飾子](./index.md)

## 概要

クラス, 関数, 変数等の要素に対してさまざまな効果の付与ができる。

## アクセス修飾子

公開範囲を指定する。

|キーワード|語源|説明|
|:-:|:-:|:-:|
|`pub`|public|要素の公開範囲を public に設定する|

### 公開範囲

デフォルトの公開範囲は private である。

#### クラスの公開範囲

|名称|対応する修飾子|公開範囲|
|:-:|:-:|:-:|
|public|`pub`|全体|
|private|-|モジュール内|

#### メンバの公開範囲

|名称|対応する修飾子|公開範囲|
|:-:|:-:|:-:|
|public|`pub`|所属するクラスと同様|
|private|-|クラス内|

## 可変修飾子

変数が可変であるかを指定する。

|キーワード|語源|説明|
|:-:|:-:|:-:|
|`mut`|mutable|変数を可変にする|
