## ホームディレクトリ

以下はOSごとのデフォルトパスである。

### Windows

```
C:\Program Files\Chestnut\
```

### macOS

```
/Users/Shared/Library/Chestnut/
```

### Linux

```
/lib/chestnut/
```

## ディレクトリ構造

```
root/
  ├ bin/
  ├ config/
  ├ docs/
    ├ vers/
```
