## バージョン管理

|名称|内容|各バージョン間の互換性|
|:-:|:-:|:-:|
|メジャーバージョン|コマンド, 構文, 機能などの大規模な追加や変更|保証なし|
|マイナーバージョン|互換性を保つ程度での機能などの追加や変更|あり|
|ビルドバージョン|環境別のバグ修正や影響の少ない細部の変更|あり|
