# 2-1. コメント

第二章からは実際にプログラムを書きながら Ches の機能や概念を習得します。

今回は特段プログラムを書きませんが、ウォーミングアップとしてコメント構文を学びましょう。

以下のように行コメントとブロックコメントを書くことができます。

```ruby
# 行コメント

#* ブロックコメント *#

#*
 * ブロック
 * コメント
 *#
```

しかしこのコメント機能はプログラムの可読性を下げる可能性があるため、特別な理由がない限り使うべきではありません。
