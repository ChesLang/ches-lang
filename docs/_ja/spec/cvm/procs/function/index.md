## 概要

## 呼び出し

配列形式である引数データの参照を関数スタック内に格納し、ポップする。

## 終了 ( return )

関数内でプッシュされた演算スタックの要素は、事前にすべてポップされる必要がある。

演算スタックに返り値をプッシュする。

関数スタック領域の例外参照データが `0xFFFFFFFF` でない場合は、参照値を取得して呼び出し元の関数スタック領域に格納する。
