## 概要

元から言語に組み込まれている関数。

言語の基礎となる機能を持つ。

## 一覧

|関数名|説明|
|:-:|:-:|
|print|コンソールへの文字列出力|
|println|コンソールへの文字列出力 ( 改行あり )|
|read|コンソールからの文字列入力 ( 1 文字 )|
|readln|コンソールからの文字列入力 ( 1 行 )|
